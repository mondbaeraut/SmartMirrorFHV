# SmartMirrorFHV

## Installation
```
npm install
```

```
sudo systemctl enable /home/pi/SmartMirrorFHV/SmartMirror.service
```

### Install Bluetooh Libraries for Noble
as described here: https://www.npmjs.com/package/noble#prerequisites

## Start
```
npm start
```
then open at http://localhost:3000
